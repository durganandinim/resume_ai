# 🤖 AI Resume Screening System

AI-powered web app that analyzes resumes using Machine Learning and NLP to predict job roles, calculate ATS score, and extract key skills.

---

## 📌 Features

- Resume upload (PDF)
- AI-based role prediction
- ATS score calculation
- Skill extraction
- Clean dashboard UI

---

## 🛠️ Tech Stack

Python, Flask, Scikit-learn, NLP (TF-IDF), Logistic Regression, HTML, CSS, PyPDF2

---

## 🧠 How It Works

Upload resume → Extract text → ML model predicts role → ATS score calculated → Skills extracted → Results displayed

---

## 📊 Example Output

- Role: Machine Learning Engineer  
- ATS Score: 90%  
- Skills: Python, SQL, TensorFlow
